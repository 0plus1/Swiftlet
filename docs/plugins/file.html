File

/_plugins/file.php

This plugin is mainly used by the /admin/files/ controller to create thumbnails for uploaded images.

Uploaded files are stored in the /uploads/files/ directory. Generated thumbnails are stored in /uploads/thumbs/

The /uploads/ folder and sub-folders need to be writable.

Class

$app->file

Public functions

  • $app->file->thumb ( string $filename, string $mimeType, string $width, string $height )
    Creates a thumbnail of an uploaded image. Thumbnails are stored in /file/uploads/thumbs/.
