MySQL

/_plugins/buffer.php

This plugin handles MySQL database connections and queries.

SQL queries

Use $app->db->sql() to execute SQL queries and $app->db->result to access the result. The type of result depends on the type of query:

  • SELECT, SHOW, EXPLAIN
    An array containing the returned rows.
  • INSERT
    The id of the inserted row.
  • UPDATE, DELETE and all others
    The number of affected rows.

Example:

db->sql('
	SELECT
		*
	FROM `' . $app->db->prefix . 'foo`
	WHERE
		`bar` = "' . $app->db->escape($bar) . '"
	;');

if ( $r = $app->db->result )
{
	print_r($r);
}
?>

Class

$app->db

Public functions

  • $app->db->close ( resource $link )
    Closes a database connection.
  • $app->db->escape ( string|array $v )
    Escapes strings for safe database insertion and HTML. Note that $app->input->POST_db_safe values are already escaped.
  • $app->db->sanitize ( string|array $v )
    Sanitizes user input (make safe for HTML and database insertion).
  • $app->db->query ( string $sql )
    Executes an SQL query.

Public variables

  • $app->db->link
    Database connection resource.
  • $app->db->result
    Returned result from query that was executed last (depending on the type of query this is either an array of returned rows, the id of the last inserted row or the number of affected rows).
