Time

Plugin to format dates/times and adjust them to the user's time zone.

Example

view->format_date(gmdate(), 'date');
?>

Class

Name
Time_Plugin
Location
/_app/plugins/Time.php

Public functions

  • format_date ( string $date [, string $type ] )
    Formats a date as $type (one of date, datetime (default) or time), adjusted to the user's time zone (default is GMT). Do not call this function directly, use $this->app->format_date() instead.

Public variables

  • $dateFormat
    Date format.
  • $timeFormat
    Time format.
  • $timeZone
    The current user's time zone (default is GMT).
  • $timeOffset
    The difference in seconds between the user's time zone and GMT.
