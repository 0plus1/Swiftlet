Time

/_plugins/time.php

Plugin to format dates/times and adjust them to the user's time zone.

Example

format_date(gmdate(), 'date');
?>

Class

$app->time

Public functions

  • $app->time->format_date ( string $date [, string $type ] )
    Formats a date as $type (one of date, datetime (default) or time), adjusted to the user's time zone (default is GMT). Do not call this function directly, use $app->format_date() instead.

Public variables

  • $app->time->dateFormat
    Date format.
  • $app->time->timeFormat
    Time format.
  • $app->time->timeZone
    The current user's time zone (default is GMT).
  • $app->time->timeOffset
    The difference in seconds between the user's time zone and GMT.
