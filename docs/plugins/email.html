E-mail

/_plugins/email.php

A plugin to send e-mail.

Example

email(
	'recipient@example.com',
	'Subject',
	'
Body
',
	array(
		'From'     => 'sender@example.com',
		'Reply-To' => 'sender@example.com'
		)
	);
?>

Class

$app->email

Public functions

  • $app->log->send ( array $params )
    Sends an e-mail. This function should not be called directly. Use $app->email() instead (see Application).
