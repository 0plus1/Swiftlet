E-mail

A plugin to send e-mail. This plugin uses PHP's mail function.

Example

app->email(
	'recipient@example.com',
	'Subject',
	'
Body
',
	array(
		'From'     => 'sender@example.com',
		'Reply-To' => 'sender@example.com'
		)
	);
?>
