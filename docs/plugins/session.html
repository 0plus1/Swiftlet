Session

This plugin manages user session data and can be used to temporarily store preferences and other information. This plugin requires a database and depends on the Database plugin.

Class

Name
Session_Plugin
Location
/_app/plugins/Session.php

Public functions

  • create ( )
    Create a new session.
  • destroy ( )
    End the current session.
  • get ( string $k [, string $default ] )
    Retrieves a session value (e.g. $this->app->session->get('user id')). If no value is set, $default is returned (optional).
  • put ( string|array $k [, string $v ] )
    Creates or updates a session value.
  • reset ( )
    Clears all session values.
  • end ( )
    Stores the session in the database and set a session cookie ("sw_session").

Public variables

  • $id
    The session id.
  • $lifeTime
    Session lifetime in seconds. The default value is 3600 (2 hours). After this period of inactivity the session will be deleted and the user will be logged out.
  • $contents
    Array containing all session values.
