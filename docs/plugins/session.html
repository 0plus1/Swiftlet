Session

/_plugins/session.php

This plug-in manages user session data and can be used to temporarily store preferences and other information. This plug-in requires a database and depends on the MySQL (or any other database) plug-in.

Class

$model->session

Public functions

  • $model->session->get ( string $k [, string $default ] )
    Use to retrieve a session value (e.g. $model->session->get('user id')). If no value is set, $default is returned (optional).
  • $model->session->put ( string|array $k [, string $v ] )
    Use to create or update a session value.
  • $model->session->reset ( )
    Use to clear all session values.
  • $model->session->end ( )
    Use to store the session in the database and set a session cookie ("sw_session").

Public variables

  • $model->session->id
    The session id.
  • $model->session->lifeTime
    Session lifetime in seconds. The default value is 3600 (2 hours). After this period of inactivity the session will be deleted and the user will be logged out.
  • $model->session->contents
    Array containing all session values.
