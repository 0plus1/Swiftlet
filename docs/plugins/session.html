Session

/_plugins/session.php

This plugin manages user session data and can be used to temporarily store preferences and other information. This plugin requires a database and depends on the MySQL (or any other database) plugin.

Class

$app->session

Public functions

  • $app->session->get ( string $k [, string $default ] )
    Retrieves a session value (e.g. $app->session->get('user id')). If no value is set, $default is returned (optional).
  • $app->session->put ( string|array $k [, string $v ] )
    Creates or updates a session value.
  • $app->session->reset ( )
    Clears all session values.
  • $app->session->end ( )
    Stores the session in the database and set a session cookie ("sw_session").

Public variables

  • $app->session->id
    The session id.
  • $app->session->lifeTime
    Session lifetime in seconds. The default value is 3600 (2 hours). After this period of inactivity the session will be deleted and the user will be logged out.
  • $app->session->contents
    Array containing all session values.
