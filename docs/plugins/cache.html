Cache

/_plugins/cache.php

Caches HTML files to improve load times and decrease server load.

Pages are cached if caching is set to TRUE in /_config.php, the user is not logged in, no Content-type headers are being send and no POST request was made. Cached files are stored in the /cache/ directory. Requires the buffer plug-in.

Class

$model->cache

Public functions

  • $model->cache->write ( string $contents )
    Writes buffered output to cache.
  • $model->cache->clear ( )
