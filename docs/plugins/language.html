Language

/_plugins/language.php

Plugin to handle translations. This plugin hooks into to the translate hook and can be called with $app->t().

Language files are placed in the /lang/ directory. Each language has its own directory named after the language with as many files as needed.

Language file:

Class

$app->lang

Public functions

  • $app->lang->check_languages ( )
    Scans the /lang/ directory for installed languages.
  • $app->lang->load ( )
    Loads a language file.
  • $app->lang->translate ( string $v )
    Translates a string (this function should not be called directly, use $app->t() instead.

Public variables

  • $app->lang->language
    The current user's language.
  • $app->lang->languages
    List of installed languages.
