Upload

This plugin is mainly used by the /admin/upload/ controller to create thumbnails for uploaded images.

Uploaded files are stored in the /uploads/files/ directory. Generated thumbnails are stored in /uploads/thumbs/

The /uploads/ folder and sub-folders need to be writable.

Name
Upload_Plugin
Location

Public functions

  • thumb ( string $filename, string $mimeType, string $width, string $height )
    Creates a thumbnail of an uploaded image. Thumbnails are stored in /uploads/thumbs/.
