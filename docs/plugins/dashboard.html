Dashboard

The dashboard provides easy access to administrative pages and displays details about your Swiftlet installation. The dashboard can be accessed at /admin/dashboard/.

This plugin comes with an additional hook named 'dashboard' for other plugins to hook into.

The Database, Permission and Session plugins are required for this plugin.

Example
 '1.3.0', 'to' => '1.3.*'),
		$hooks      = array('dashboard' => 1)
		;

	function dashboard(&$params)
	{
		$params[] = array(
			'name'        => 'Example',
			'description' => 'Example page',
			'group'       => 'Examples',
			'path'        => 'admin/example/',
			'permission'  => 'admin example access'
			);
	}
}
?>

To add a link to the dashboard, use the dashboard hook with the following parameters:

  • name
    The name of link as shown on the dashboard.
  • description
    A brief description of the link, ideally not more than a few words.
  • group
    Links using the same group name a grouped together.
  • path
    Path of the link, relative to the root of the website.
  • permission
    Links will only be shown if the user has the required permission (see Permsission).
