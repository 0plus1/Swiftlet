Installation

Installing Swiftlet is as easy as extracting the archive and putting the files on your web server. Browse to the Swiftlet directory in your browser and you should be greeted with a welcome message.

The next step is the create a configuration file by copying /_config.default.php to /_config.php. Review the new file and make changes if desired.

If you configured a database in /_config.php you can now active some or all of the bundled plugins that require installation by going to /installer/.

Prerequisites

  • A web server (e.g. Apache)
  • PHP 5.1 or higher

A MySQL database server is optional but required by some plugins.
