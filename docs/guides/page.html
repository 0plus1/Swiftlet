Creating pages

The easiest way to add a page to your website is using the Page plugin. After installation you'll find a link on the Dashboard where you can add and edit pages. These pages can only contain static content such as text and images and are useful for basic documents and articles.

To add a dynamic (scripted) page you need to create two files, a Controller and a View.

Optional properties can be specified as public variables, such as $pageTitle.

  • $dependencies
  • $pageDescription
    Description of the page, use for the 'description' meta tag.
  • $pageKeywords
    A list of keywords relevant to the page, use for the 'keywords' meta tag.
  • $pageTitle
    Title of the page. This value will be displayed in the web browser's title bar.
  • $standAlone
    If TRUE, the header and footer views will not be included. This is useful for pages such as RSS feeds that don't need to be wrapped in the website's main layout.
  • $inAdmin

At this point you can add your own PHP code.

Now you can load the View file with $view->load(). This file will contain all the HTML and other UI elements. This way logic is separated from design which will make it easier to maintain.

When you browse to the Controller in your web browser (?q=/hello or /hello if URL rewrites are enabled) the page should appear.
