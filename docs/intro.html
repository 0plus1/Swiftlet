Swiftlet is a light-weight web application framework written in PHP. It enables you to quickly deploy websites that are fast and secure.

This documentation is aimed a developers who want to get started with Swiftlet or extend it. It explains everything you need to know from installation to creating custom plugins.

If you're new to Swiftlet you might want to start with the Guides section. The articles in the Concepts section will help you understand Swiftlet's architecture.

For support, visit http://swiftlet.org.
