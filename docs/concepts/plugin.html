Plugins

Plugins are programs that extend Swiftlet's functionality. They can be installed without modifying any of the original code.

Plugins are stored in the /_app/plugins/ directory. Swiftlet will scan this directory and include every plugin it finds. Plugins that require a database need to be installed using the plugin installer at /installer/.

Each plugin is a class that extends the 'Plugin' class (described below). An instance of the class is automatically created when a plugin is loaded and can be access through the Application (e.g. $this->app->{plugin_name}).

To learn how to write plugins, see Creating plugins.

Public variables

  • $class
    The name of the plugin class.
  • $compatible
    The lowest and highest version numbers of Swiftlet that this plugin is compatible with. A wildcard character (*) may be used to specify multiple versions.
  • $dependencies
    An array containing the names of plugins that are required for the plugin to work.
  • $file
    The filename of the plugin.
  • $hooks
    List of hooks the plugin utilizes and the order in which they should be executed.
  • $installed
    The installed version number of the plugin, or FALSE if the plugin is not installed.
  • $name
    The name of the plugin.
  • $upgradable
    List of plugin version numbers that are upgradable to the current version.
  • $version
    Version number of the plugin.
