Helpers

Helpers are snippets of code that can be called in Views to help with tasks and keep your code DRY. Any code that you are repeating regularly can likely be moved into a helper.

Helpers are stored in /_views/_helpers/.

Example

helper('pagination', array('name' => 'users', 'page' => 1, 'pages' => 3));

?>

The helper function takes two arguments; the name of the helper you want to call and optionally an array of parameters. In the example's case the file /_views/_helpers/pagination.hmtl.php will be included to display pagination.
