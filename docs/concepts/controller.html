Public variables

  • $inAdmin
    Used for administrative pages to differentiate between normal pages.
  • $pageDescription
    Description of the page.
  • $pageKeywords
    Keywords relevant to the contents of the page.
  • $pageTitle
    Title of the page.
  • $standAlone
    If this variable is set to TRUE the header and footer hooks will not be executed.

Protected variables

  • $action
    Action specified in the request (see Routes).
  • $app
    Refers to the main instance of the Application class.
  • $dependencies
  • $id
    The id specified in the request (see Routes).
  • $path
    Path to the current controller (see Routes).
  • $request
    Path to the current controller, including arguments (see Routes).
  • $view
    Refers to the main instance of the View class.
