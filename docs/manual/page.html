Pages

This page is about creating dynamic (scripted) pages. To learn about adding static content, see the documentation for the Page plug-in.

  1. To create a new page, create a Controller (PHP file) and View (HTML file) as described on the linked pages. The Controller in the root (/) or a sub directory and the View in /_view/.

    The reason for the two files is to seperate logic from presentation. You always need to create two files, even if the page only contains static content.
  2. The next step is to make sure the correct root path is set in the Controller. This should be a relative path from the Controller to the root of the installation (e.g. ./ or ../../). The root path is defined in the $contrSetup array on top of the file (rootPath) and can never be empty.
  3. Check if the View file is linked correctly (e.g. $view->load('index.html.php'). The path should be relative to the /_view/ directory.
  4. If needed you can add PHP code to the Controller. To pass variables to the View, the $view object should be used (e.g. $view->foo = 'bar'. All static content (text, images etc.) goes into the View file.
  5. Open the Controller file in your web browser to see the page.
